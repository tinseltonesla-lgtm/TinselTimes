How to Embed the Caroler Portal

To add the TinselTimes Caroler Portal to your website, follow these two steps:

1. Add the Placeholder

Paste this where you want the portal to appear:

<div id="tinseltimes-portal"></div>

2. Add the JavaScript Code

Paste this script tag before your closing </body> tag:

<script src="tinseltimes-embed.js"></script>
FIXING THE "FORBIDDEN" ERROR:
1. Ensure tinseltimes-embed.js and index.html are uploaded to the same folder on your web server.
2. Do not open these files directly from your computer folder (double-clicking the file). You must view them through a URL (e.g., https://yoursite.com/portal.html).
3. If using a service like GoDaddy or Wix, ensure "Iframe Embedding" is allowed in your security settings.
